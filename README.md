## Mistakes & Lessons Learned
- cd.. needs a space → correct is cd ..
- wc -1 is wrong → correct is wc -l (letter L)
- echo without > prints to screen not file
- Folders with spaces cause errors → use - or _
- > overwrites file, >> appends to file
